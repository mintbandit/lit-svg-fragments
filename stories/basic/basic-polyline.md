# Example Polyline Structure

```html
<svg height="100" width="100">
  <polyline
    ...polyline-attributes      
  />
</svg>
```

## Basic Polyline

<basic-polyline svg-height="100" svg-width="100" points="50,0 21,90 98,35 2,35 79,90"></basic-polyline>

```html
<svg height="100" width="100">
  <polyline points="50,0 21,90 98,35 2,35 79,90" />
</svg>
```

> The most basic SVG polyline visually will look identical to an SVG polygon. The differences become clear when additional
> attributes are applied.
